# 小组项目

## 仿猫眼电影web移动端应用			

## 			React+Redux+React-redux+Antd-mobile



## 1.将项目下载至本地，并补全依赖

```
git clone https://github.com/tomtmao/movieap.git
yarn install | npm install | cnpm install
```

## 2.启动项目

```
yarn start | npm start | cnpm start
```
